# python3
- Serveix per a: Executar scripts escrits en Python (versió 3).
- Exemple:**python3 script.py**
# cc
- Serveix per a: Compilar programes escrits en C. És una abreviació per al compilador de C, habitualment vinculat a GCC (GNU Compiler Collection).
- Exemple:**cc programa.c -o programa**
# java
- Serveix per a: Executar programes escrits en Java.
- Exemple:**java NomDelPrograma**
# javac
- Serveix per a: Compilar el codi font escrit en Java en bytecode que pot ser executat per la màquina virtual de Java (JVM).
- Exemple:**javac NomDelPrograma.java**
